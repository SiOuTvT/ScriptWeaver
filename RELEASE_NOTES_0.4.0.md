# ScriptWeaver v0.4.0 发行说明

**涵盖范围**：v0.3.0 至今的全部新增、改进与修复

## ✨ 新增功能

### 1. 特效大本营 可挂载特效三大类目
把「特效大本营」从纯展示升级为可挂载到时间轴、可导出 Ren'Py 脚本的实用系统，并重组为三大类目，转场与滤镜一并纳入。

- 组件 / 元素特效（Element）：作用于单个立绘或背景，导出为 at &lt;transform&gt; 叠加。
  - 自定义抖动、弹性放大、描边闪烁、旋转、整体缩放、模糊、透明度，新增呼吸效果（Breathing）、位置微调（Nudge）。
- 全屏转场（Transitions）：作用于场景切换或剧本行行进，导出为 with &lt;transition&gt;。
  - 内建工厂直出：溶解（Dissolve）、像素化（Pixellate）、淡入淡出（Fade）、四向擦除（Wipe，右 / 左 / 上 / 下）。
  - 自定义 ATL：水平 / 垂直震屏、闪白、百叶窗。
- 全屏滤镜（Filters）：作用于整层色调氛围，导出为 show layer master: matrixcolor &lt;Matrix&gt;。
  - 新增黑白 / 回忆（Monochrome）、老照片（Sepia）、调色（ColorMatrix / Tint，如血腥红光、中毒绿等），复用 Ren'Py 内建矩阵，多滤镜可叠加，关闭后自动还原正常色调。
- 时间轴挂载面板下拉按三大类目分组，已挂载项显示类目标签（元素 / 转场 / 滤镜），参数可实时微调。

### 2. Ren'Py 导出设置改进
- 多导出目标：Windows / macOS / 移动端 / Web 端切换与打包偏好。
- 导出流程可视化：校验引用 → 生成脚本 → 生成定义 → 打包素材 → 完成，分阶段推进，校验不通过会明确提示。
- 项目包一键导出：生成完整 game/ 目录（含 script.rpy / definitions.rpy / images / audio），放入 Ren'Py 工程即可运行。
- 脚本入口 label 可配置；导出前展示项目概况（剧本行数 / 出场说话人 / 角色配置）。

### 3. 内置 AI 导演
- 密钥在本地安全托管，界面只加载必要配置，不会泄露。
- 每次请求设总超时与断流保护，报错以中文分级提示，杜绝界面卡死或空白。
- 支持流式输出、多服务商预设切换，AI 指令自动解析为画面与音频编排。

### 4. 外观与主题色系统
- 预设主题、明暗模式、色相微调、自定义色号（#RGB / #RRGGBB 即时预览），浅 / 深双语境实时预览，保存后全站按钮、链接、焦点、信号色同步。

### 5. 特效百科专业副栏
- 特效详情页右侧新增推荐动效组合（音效 / 转场 / 镜头 / 特效 / 节奏）、经典 Galgame 名场面参考、常见错误用法对照；无精选内容时也会自动补充分类，右栏不空白。

## 🔧 改进

### 字体与可读性
- 拉丁字体统一 Noto Sans + 中文 Noto Sans SC 同族，字重收敛至 400 / 500 / 600。
- 全站字号统一到 12 / 13 / 14 / 15 / 16 / 18 档，正文 14px，下限 12px。
- 浅色模式分层更清晰：画布底色更深、各面板层级拉开、弱文字对比度加深，小字更易读；深色模式层级同步优化。

### 舞台与交互
- 立绘改为拖动调整位置、双击才打开属性面板，避免误触；选中元素后按 Delete / Backspace 可直接删除，并同步更新时间轴。
- 多立绘独立管理，互不覆盖、互不干扰。

### 资产管理
- 素材改为统一协议管理，项目文件只记录引用而非内嵌大体积数据，更安全、体积更小，也避免路径泄露。
