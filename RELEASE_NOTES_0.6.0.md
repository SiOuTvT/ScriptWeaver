# ScriptWeaver v0.6.0 发行说明

**发布日期**：2026-07-21
**发布主题**：抽屉式布局 | 沉浸式舞台 | 安全增强 | 视觉重构

---

## 新增功能

- **抽屉式右侧检视面板**：剧本流、变量监视、选择支等检视面板改为浮层抽屉——需要时从右侧滑出，不用时自动收起。不再常驻挤占舞台空间，每个抽屉可自由拖拽调整宽度，随心布局。
- **剧本总览双视图**：大纲视图（结构化剧本树）与节点卡片视图（可视化节点网）一键切换。顶部仪表盘实时统计总字数、场景数、分支数、角色数，过滤栏粘顶提供沉浸式筛选体验。
- **自动化版本快照**：每次保存自动备份项目快照，保留多达 60 个历史版本，随时可回滚到任意快照。再也不怕误改丢失创意。

## 体验改进

- **舞台预览大幅释放**：右侧面板默认收起，舞台与底部时间轴拉满全屏，营造纯粹的沉浸式剧本编辑环境。
- **条件分支可视化追踪**：时间轴中条件分支以醒目背景线连接，点击即可定位到对应选择支节点，剧本逻辑路径一目了然。
- **素材管理体验升级**：瀑布流卡片悬停时立体抬升、边框高亮，分类栏加入渐变高光条，视觉层次更分明；音频波形与时间轴播放器、PNG 棋盘格底衬等多媒体预览功能均保持流畅。
- **角色管理名册抛光**：角色卡片悬停反馈更细腻，左侧名册、右侧表情墙的主从布局交互流畅度持续优化。
- **主题切换更丝滑**：浅色/深色主题过渡动画优化，所有按钮与卡片高亮色系统一，换肤体验更连贯。

## Bug 修复

- **修复表达式求值的潜在安全隐患**：变量表达式解析升级为更安全的处理方式，杜绝恶意数据利用的风险，同时确保历史剧本逻辑完全兼容。
- **修复加载部分项目时崩溃**：增强项目文件读取的容错能力，素材或音频配置不完整时不再报错退出，自动补全安全默认值。
- **修复删除素材后的内存问题**：删除素材时自动释放对应缓存，长时间编辑后内存使用更稳定，避免积压。
- **修复 Ren'Py 导出兼容性问题**：
  - 非法变量名（含多音字、关键字）自动清洗为合法标识符，导出不再报错。
  - 对话文本含换行符导致剧本断裂——自动转义，确保导出的 Ren'Py 剧本能正常编译运行。
  - 角色名与颜色包含引号时语法错误——引号统一转义，角色定义不再断裂。
  - 内置过渡特效列表与最新 Ren'Py 官方手册对齐：移除已废弃的过渡名称，补全缺失的标准特效。
- **修复变量监视面板频闪问题**：更新频率优化，变量值变化时面板不再频繁闪烁。
