# ScriptWeaver v0.5.0 发行说明

发布日期：2026-07-21
适用升级：从 v0.4.0 升级
涵盖范围：v0.4.0 至今的全部新增、改进与修复

---

## 新增功能

### 1. 可停靠的工作台
整体改为 IDE 式布局：中央舞台占据主区域，底部时间轴可折叠，素材库、剧本、角色、变量、选择支等面板环绕四周，可收拉与停靠，把最大空间留给创作核心，不再用浮层遮挡舞台的常用操作。

### 2. 角色管理
新增独立的角色管理：维护角色名册与详情，可设置头像、显示名与变量名、主题色，绑定声色并试听，配置默认出场缩放与槽位，以及管理角色表情包（设默认、重命名、删除、从素材库添加、拖入系统图片自动绑定）。

### 3. 剧本总览
新增剧本总览面板：顶部实时统计字数、剧情块、选择支、角色等概况；支持卡片网格与大纲两种视图切换；可按文本、角色与维度筛选；点击任意卡片或行可跳转回场景并高亮定位。

### 4. 素材管理
素材库改为分类树与工具栏组织：按类别浏览并带计数；支持即时搜索、排序、网格大小与列表视图切换；音频素材带波形播放器可试听与跳转；可直接从系统拖入图片或音频落盘导入；可检索某素材被哪些剧本行引用。

### 5. 从系统拖入素材
支持把系统文件管理器里的图片或音频直接拖进软件完成导入，无需额外对话框。

## 改进

### 舞台与交互
立绘可拖到舞台上摆放，双击打开属性面板避免误触，选中后按删除键直接移除并同步时间轴。多立绘各自独立管理，互不覆盖、互不干扰。

### 可读性与外观
延续统一字体与字号档位，浅色与深色模式的分层与文字对比继续优化，弱文字也保证清晰可读。

## 修复
- 修复素材文件缺失或加载失败时，舞台只显示空白、没有任何提示的问题；现在会显示带角色名的占位并标明素材缺失，便于定位是哪一张素材丢了。
- 修复拖拽立绘到舞台后偶发不显示的问题，提升落点解析与渲染的健壮性。
