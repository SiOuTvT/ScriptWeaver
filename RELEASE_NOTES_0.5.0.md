# ScriptWeaver v0.5.0 发行说明

发布日期：2026-07-21
适用升级：从 v0.4.0 升级
涵盖范围：v0.4.0 至今的全部新增、改进与修复

---

## 新增功能

### 1. 选择支编辑器
在时间轴选中选择支行时，右侧属性面板会自动切换为选项编辑器：可新增或删除选项、为每个选项指定跳转到哪个剧情块、给单个选项设置变量增减操作，并实时预览变量操作的效果。

### 2. 剧情节点图谱
把带有标记的剧情块绘制成节点卡片，选择支的跳转关系自动画成节点之间的连线。可拖动节点、点击节点或连线跳到对应剧本位置、直接拖动连线末端重新绑定跳转目标；节点按跳转关系自动分层排布。

### 3. 变量监视与调试器
列出版本里定义的全部全局变量当前运行值；剧本播放或做出选择时，数值会实时变化并高亮提示；创作者可直接手动修改当前值，方便测试后续分支走向。调试时的修改只在预览态生效，不影响最终导出。

### 4. 时间轴特效挂载
在舞台属性面板中，可以为当前剧本行挂载特效：从三大类目（元素特效、全屏转场、全屏滤镜）中选取预设，每个已挂载特效可独立开关并微调参数，实时查看效果变化。

### 5. 特效百科智能推荐
查看某个特效的详情时，右侧自动展示：该特效的推荐搭配组合（配什么音效、衔接什么转场、搭什么镜头运动）并给出具体理由；经典 Galgame 中该特效的使用场景参考；以及常见错误用法与正确用法的避坑对照。

### 6. 从系统拖入素材
支持把系统文件管理器里的图片或音频直接拖进软件完成导入，无需额外的对话框。

## 改进

### 整体改为可停靠的工作台
界面重做为 IDE 式可停靠布局：中央舞台占据主区域，底部时间轴可折叠，素材库、剧本、角色、变量、选择支等面板环绕四周可收拉停靠，把最大空间留给创作核心，不再用浮层遮挡舞台的常用操作。

### 角色管理重做
维护角色名册与详情，可设置头像、显示名与变量名、主题色，绑定声色并试听，配置默认出场缩放与槽位，以及管理角色表情包（设默认、重命名、删除、从素材库添加、拖入系统图片自动绑定）。

### 剧本总览重做
顶部实时统计字数、剧情块、选择支、角色等概况；支持卡片网格与大纲两种视图切换；可按文本、角色与维度筛选；点击任意卡片或行可跳转回场景并高亮定位。

### 素材管理重做
素材库改为分类树与工具栏组织：按类别浏览并带计数；支持即时搜索、排序、网格大小与列表视图切换；音频素材带波形播放器可试听与跳转；可检索某素材被哪些剧本行引用。

### 特效大本营增强
特效大本营现在是一个完整的展示大厅：按分类浏览数十种内置特效，在预览区实时观看效果动画，支持搜索与筛选。每个特效的详情页附带使用说明、参数解读、搭配建议与经典参考。

### 导出设置重做
导出流程改为多平台多阶段看板：可选择目标平台（Windows / macOS / 移动端 / Web 端），分阶段执行校验引用、生成脚本、生成定义、打包素材、完成，每阶段有明确的状态反馈；导出前可自定义脚本标签名与导出选项。

### AI 导演增强
新增语义标签精准匹配：AI 返回的角色、背景、音效标签能稳定命中素材库中的真实素材。导演模式下 AI 可自动生成完整剧本行并完成角色站位、表情、背景、BGM 的自动打点。支持多厂商切换（OpenAI / DeepSeek / OpenRouter / 自定义端点），密钥安全托管不变。

### 导出器增强
挂载到时间轴的特效现在可完全闭环导出为相应的脚本语句，自定义特效自动生成参数化变换定义并写进独立文件；音频按四通道隔离输出（BGM 进音乐通道、环境音自注册通道、音效进声音通道、语音进语音通道）；内建过渡直接使用原生语法无需额外定义；新增一键项目打包，生成工程目录即可放入对应工具运行。

### 舞台立绘交互
立绘可拖到舞台上摆放，双击打开属性面板避免误触，选中后按删除键直接移除并同步时间轴。多立绘各自独立管理，互不覆盖、互不干扰。

### 可读性与外观
延续统一字体与字号档位，浅色与深色模式的分层与文字对比继续优化，弱文字也保证清晰可读。

## 修复
- 修复素材文件缺失或加载失败时，舞台只显示空白、没有任何提示的问题；现在会显示带角色名的占位并标明素材缺失，便于定位是哪一张素材丢了。
- 修复拖拽立绘到舞台后偶发不显示的问题，提升落点解析与渲染的健壮性。
